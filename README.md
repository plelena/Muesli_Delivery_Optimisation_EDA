# Muesli_EDA

This repository contains EDA on Muesli store data. Steps included are:
  - Data sourcing
  - Data cleaning
  - know your Data
  - Hypothesis creation
  - Data aggregation
  - Data Augmentation
  - Data Visualizations
  - Hypothesis acceptance or rejection
  - KPI suggestions
  - Stakeholder presentation
  
  Tools used: Jupyter Notebook, Python, Pandas, Matplotlib, Seaborn
